## Quick start (local)

1. Install Java 21 and Maven
2. Set `JAVA_HOME` to your JDK 21 installation
3. Build:

```powershell
$env:JAVA_HOME = 'C:\Program Files\Java\jdk-21'
$env:Path = "$env:JAVA_HOME\bin;" + $env:Path
mvn -U clean package
```
